
# 📊 Customer Review Analyzer

A professional Streamlit application that analyzes customer reviews using **sentiment analysis**.  
It classifies words as **positive (blue)** or **negative (red)** and provides interactive visualizations:

- 📈 **Sentiment Bar Chart** – Frequency of positive vs negative words  
- ☁️ **Sentiment Word Cloud** – Highlights positive and negative words separately  
- 🔗 **Sentiment Word Network** – Connects related positive and negative words  

---

## 🚀 Features
- Text preprocessing with **spaCy** (lemmatization, stopword removal)  
- Sentiment classification using **NLTK opinion lexicon**  
- Interactive dashboard built with **Streamlit**  
- Consistent color scheme: **blue for positive**, **red for negative**  
- Clean, spacious UI for professional presentation
